# AvatarDuplicator

## Copy Target
* Avatar (On Scene)
* FX Layer
  * Animation Clip (Optional)
* ExpressionsMenu
* ExpressionParameters

## How to Use

1. Select [AvatarDuplicator/Duplicate] from the Menu.
2. Set avatar to Source Object.
3. (Optional) Toggle copy animationClip option.
3. Press [Duplicate] Button.
4. Done.
